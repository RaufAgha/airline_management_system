# Airline Ticket Management System

## Overview
This is a Python-based Airline Ticket Management System developed using **Object-Oriented Programming (OOP)** principles.  
The system allows management of **Flights**, **Passengers**, **Tickets**, and **Aircrafts** with full CRUD functionality.  

The project demonstrates:  
- OOP principles: **Abstraction, Encapsulation, Inheritance, Polymorphism**  
- SOLID, GRASP, and CUPID design principles  
- Layered architecture: **Models → Repositories → Services → CLI**

---

## Features

- Add, view, delete **Flights**
- Add, view, delete **Passengers**
- Add, view, delete **Tickets**
- Add, view, delete **Aircrafts**
- Prevent duplicate bookings
- Validate flight and passenger existence
